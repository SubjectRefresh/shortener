API docs

The Shortener HTTP API is very simple. There are two endpoints, as shown below - one for creating short links and one for finding out where they lead (lengthening)

Endpoint Method Example data object Example response
/api/create POST { url: "http://yourdomain.com" } { status: true, shortlink: "http://subr.pw/9zckSc"}
/api/lengthen POST { url: "9zckSc" } { status: true, url: "http://yourdomain.com" }
